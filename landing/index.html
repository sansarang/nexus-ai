FAQ

자주 묻는 질문

Groq API는 무료로 발급됩니다 (groq.com). 가입 후 1분이면 API 키를 받을 수 있습니다. GPT-4o를 사용하고 싶다면 OpenAI API 키가 필요합니다.
모든 처리는 127.0.0.1(로컬)에서 이루어집니다. AI 응답 생성 시 Groq/OpenAI API로 질문 내용이 전송되지만, PC 파일·이메일 내용·시스템 정보는 외부로 전송되지 않습니다.
개인 사용 목적의 가격 비교·정보 수집은 일반적으로 허용됩니다. 상업적 대량 수집이나 로그인 우회 등은 각 사이트 이용약관을 확인하세요.
Windows 10 이상을 권장합니다. PC 조작 기능(PowerShell, WMI, Outlook COM)은 Windows 전용입니다. Mac은 기본 AI 대화 기능만 지원됩니다.
GitHub Releases를 통해 자동 업데이트됩니다. 앱 실행 시 새 버전이 있으면 알림이 뜨고, 한 번의 클릭으로 최신 버전으로 업데이트됩니다.
Outlook 자동화(발송·일정 연동)는 Outlook 설치가 필요합니다. 네이버·다음·카카오 메일은 IMAP 설정으로 읽기 기능을 사용할 수 있습니다.
