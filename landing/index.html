HOW IT WORKS

3분이면 시작합니다

1

설치 & 구글 로그인 (3분)

15MB 설치 파일 실행 후 Google 계정으로 로그인만 하세요. API 키 발급·등록은 저희가 처리합니다. 직업군 페르소나를 선택하면 즉시 사용 가능합니다.

2

말하거나 입력하기

Alt+Space 단축키로 어디서든 NEXUS를 호출합니다. 텍스트 입력 또는 "넥서스야" 웨이크워드로 명령하세요. 파일 첨부, 모드 전환, 워크플로우 설정 모두 가능합니다.

3

NEXUS가 실행합니다

PC 최적화, 파일 분석, 메일 발송, 웹 크롤링 등 실제 작업을 수행하고 결과를 보고합니다. 자동 업데이트로 항상 최신 기능을 유지합니다.

FAQ

자주 묻는 질문
