3분이면 시작합니다

1

3

NEXUS가 실행합니다

PC 최적화, 파일 분석, 메일 발송, 웹 크롤링 등 실제 작업을 수행하고 결과를 보고합니다. 자동 업데이트로 항상 최신 기능을 유지합니다.

FAQ

자주 묻는 질문

전혀 없습니다. Google 계정으로 로그인만 하면 API 키 발급과 등록은 저희가 자동으로 처리합니다. 설치부터 첫 AI 응답까지 3분이면 됩니다.
모든 처리는 127.0.0.1(로컬)에서 이루어집니다. AI 응답 생성 시 질문 내용만 Groq/OpenAI API로 전송되며, PC 파일·이메일 내용·시스템 정보는 절대 외부로 전송되지 않습니다.
Early Access 사용자 설문 기준입니다. 반복 업무(이메일 정리, 엑셀 분석, 가격 비교, PC 최적화)를 자동화하면 업무 성격에 따라 1~4시간 절약이 가능합니다. 3일 무료 체험으로 직접 확인해보세요.
개인 사용 목적의 가격 비교·정보 수집은 일반적으로 허용됩니다. 상업적 대량 수집이나 로그인 우회 등은 각 사이트 이용약관을 확인하세요.
Outlook 자동화(실제 발송·일정 연동·스마트 답장)는 Outlook 설치가 필요합니다. Gmail·네이버·다음은 IMAP으로 읽기·요약을 지원하며, 설정에서 이메일 주소와 앱 비밀번호를 입력하면 됩니다.
현재 Windows 10/11이 메인 지원 플랫폼입니다. Mac에서는 AI 대화, 웹 검색, 브리핑 등 기본 기능은 사용 가능하지만 PowerShell, WMI, Outlook COM 등 Windows 전용 기능은 지원되지 않습니다.
