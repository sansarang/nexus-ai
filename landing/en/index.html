FAQ

Frequently Asked Questions

Groq API is free (groq.com). Sign up and get your API key in under 1 minute. For GPT-4o, you'll need an OpenAI API key separately.
All processing happens at 127.0.0.1 (local). Only your query text is sent to Groq/OpenAI for AI responses — your files, emails, and system data never leave your PC.
Personal-use price comparison and information retrieval is generally permitted. Commercial bulk scraping or login bypassing should comply with each site's Terms of Service.
Windows 10 or later is recommended. PC control features (PowerShell, WMI, Outlook COM) are Windows-exclusive. Mac supports basic AI chat only.
Updates are delivered via GitHub Releases using go-selfupdate. On launch, NEXUS checks for a new version and prompts you — one click updates to the latest.
Full automation (send, reply, calendar sync) requires Outlook installed. Naver, Daum, and Kakao Mail support read & summary only via IMAP — auto-send is Outlook-only.
