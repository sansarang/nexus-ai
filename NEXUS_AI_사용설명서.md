## 1. NEXUS AI란?

NEXUS AI는 Windows PC를 AI가 직접 운영해주는 **로컬 실행형 AI 비서**입니다.

| 구분 | 내용 |
|---|---|
| 동작 방식 | Windows 네이티브 앱 (Tauri + Go 백엔드) |
| 실행 위치 | 사용자 PC 내부 (127.0.0.1) |
| 데이터 보안 | PC 파일·이메일·시스템 정보는 외부 전송 없음 |
| AI 엔진 | Perplexity / Claude (선택) |
| 지원 OS | Windows 10 (1903 이상) / Windows 11 |

### NEXUS AI가 할 수 있는 것

- 말 한마디로 PC 최적화, 임시파일 정리, 드라이버 업데이트
- 쿠팡·네이버·다나와 실시간 가격 비교 (실제 크롤링)
- Excel·PDF·Word 파일 열기 → AI가 분석·수정·저장
- Outlook 메일 자동 요약·답장·발송
- 매일 아침 날씨·일정·PC 상태 자동 브리핑
- 반복 업무를 No-code 워크플로우로 자동화

## 2. 설치 및 첫 실행

### 설치 과정

1. **다운로드**
   - GitHub Releases에서 `Nexus_x64-setup.exe` 다운로드
   - https://github.com/sansarang/nexus-ai/releases

2. **설치 실행**
   - `Nexus_x64-setup.exe` 더블클릭
   - "이 앱이 변경을 허용하시겠습니까?" → **예** 클릭
   - 설치 경로 선택 (기본값 권장): `C:\Users\사용자명\AppData\Local\Nexus`
   - **설치 완료** 클릭

3. **첫 실행**
   - 바탕화면 또는 시작 메뉴에서 **Nexus AI** 실행
   - 시스템 트레이(우측 하단 시계 근처)에 NEXUS 아이콘이 나타남
   - 화면 우측에 AI 캐릭터(아바타)가 등장

> **참고**: 첫 실행 시 Windows Defender 경고가 뜰 수 있습니다.  
> "추가 정보" → "실행" 을 클릭하세요.

## 3. API 키 설정

NEXUS AI는 AI 엔진으로 **Perplexity API**를 사용합니다. 무료로 시작할 수 있습니다.

### 3-1. Perplexity API 키 발급 (무료)

1. https://www.perplexity.ai 접속 → 회원가입
2. 우측 상단 프로필 → **API** 메뉴 클릭
3. **+ Generate** 클릭 → API 키 복사 (`pplx-...` 로 시작)

### 3-2. NEXUS에 API 키 입력

1. NEXUS 채팅창 상단 ⚙️ **설정** 버튼 클릭
2. **AI 엔진 설정** 탭
3. **Perplexity API 키** 입력란에 붙여넣기
4. **저장** 클릭

### 3-3. 추가 API 키 (선택)

| API | 용도 | 발급처 |
|---|---|---|
| Tavily API | 웹 검색 기능 강화 | https://tavily.com |
| Claude API | AI 엔진 대체/보조 | https://console.anthropic.com |

> **보안**: 입력한 API 키는 Windows DPAPI로 암호화하여 로컬 저장합니다.  
> 외부 서버로 전송되지 않습니다.

## 5. 아바타 캐릭터 선택

4가지 개성 있는 AI 캐릭터 중 하나를 선택할 수 있습니다.

| 캐릭터 | 이름 | 음성 | 특징 |
|---|---|---|---|
| 🧑‍🚀 | 우주비행사 아리 | 밝고 경쾌 (Nova) | 친근한 일상 도우미 |
| 🐯 | 타이거 맥스 | 중후하고 전문적 (Onyx) | 분석·업무 특화 |
| 🦋 | 숲속 친구 나비 | 부드럽고 따뜻 (Shimmer) | 창의적·감성적 |
| 🦅 | 자유의 날개 노바 | 에너지 넘침 (Alloy) | 엔터테인먼트·콘텐츠 |

### 캐릭터 변경 방법

1. 캐릭터를 **우클릭** 또는 채팅창 상단 **캐릭터 아이콘** 클릭
2. 원하는 캐릭터 선택
3. 즉시 적용 (재시작 불필요)

> **참고**: 음성(TTS) 기능은 OpenAI TTS API가 설정된 경우 캐릭터별 목소리가 달라집니다.  
> API 미설정 시 Windows 기본 음성이 사용됩니다.

## 6. AI 대화 (기본 채팅)

### 채팅창 열기

캐릭터를 **클릭**하거나 시스템 트레이 아이콘을 클릭하면 채팅창이 열립니다.

### 대화 방법

채팅 입력창에 질문을 입력하고 **Enter** 키를 누릅니다.

**예시 명령어**

```
"오늘 서울 날씨 알려줘"
"파이썬으로 피보나치 수열 코드 짜줘"
"지금 PC 상태 어때?"
"쿠팡에서 아이패드 최저가 찾아줘"
"내 받은 메일 5개 요약해줘"
```

### 채팅창 크기 조절

채팅창 **왼쪽 상단 모서리**를 드래그하면 크기를 자유롭게 조절할 수 있습니다.
- 최소: 가로 260px × 세로 300px
- 최대: 가로 600px × 세로 700px

### 한국어 입력 주의

한글 입력 시 **마지막 글자가 두 번 입력**되는 문제를 방지하기 위해 IME 조합 중 Enter 무시 기능이 적용되어 있습니다. 자연스럽게 입력하시면 됩니다.

## 7. AI 모드 (페르소나) 전환

NEXUS AI는 상황에 맞게 **5가지 AI 모드**를 제공합니다.

### 모드 종류

| 모드 | 아이콘 | 특화 기능 |
|---|---|---|
| **Nexus 기본** | 🤖 | 모든 기능 균형 있게 사용 |
| **전문가 모드** | 🧠 | 딥서치·학술 자료 우선, 10개+ 소스 분석 |
| **리서치** | 🔬 | 경쟁사 분석, 시장 조사 특화 |
| **크리에이티브** | 🎨 | 아이디어 발상, 콘텐츠 기획 |
| **재무** | 💰 | 예산 분석, 재무 보고서 작성 |

### 모드 전환 방법

1. 채팅창 오른쪽 사이드바의 **🧠 버튼** 클릭
2. 원하는 모드 선택
3. 즉시 적용 (현재 활성 모드가 상단에 표시됨)

### 전문가 모드 특징

전문가 모드 활성화 시:
- 웹 검색 결과를 **10개 이상** 분석
- arxiv, scholar, pubmed 등 학술 데이터베이스 우선 검색
- 데이터·근거 기반 심층 답변 제공
- 채팅창 하단에 🧠 안내 배너 표시

## 8. 파일 첨부 & 문서 AI 분석·편집

NEXUS AI는 파일을 첨부하면 **분석** 또는 **편집**을 자동으로 판단합니다.

### 지원 파일 형식

| 형식 | 분석 | 편집·저장 |
|---|---|---|
| Excel (.xlsx, .xls) | ✅ | ✅ (excelize 기반) |
| Word (.docx) | ✅ | ✅ |
| CSV | ✅ | ✅ |
| PDF | ✅ | — (읽기 전용) |
| 이미지 (jpg, png) | ✅ (Vision AI) | — |
| 텍스트 (.txt, .md) | ✅ | ✅ |

### 파일 첨부 방법

1. 채팅 입력창 왼쪽 **📎 버튼** 클릭
2. 파일 선택 (또는 드래그 앤 드롭)
3. 지시사항 입력 후 **Enter**

### 분석 vs 편집 — 자동 판단

NEXUS AI는 입력 내용에 따라 자동으로 모드를 결정합니다.

**분석 모드로 처리되는 예시**
```
📎 매출현황.xlsx 첨부 후:
"이 데이터 분석해줘"
"어떤 제품이 가장 많이 팔렸어?"
"인사이트 정리해줘"
```

**편집 모드로 처리되는 예시**
```
📎 매출현황.xlsx 첨부 후:
"매출 기준으로 내림차순 정렬해줘"           → sort 연산 적용
"3번 행 삭제해줘"                           → delete_rows 적용
"D2셀에 =SUM(A2:C2) 수식 넣어줘"           → formula 적용
"헤더 행에 노란 배경색 적용해줘"             → style 적용
"B열 기준으로 TOP 10만 남겨줘"             → 필터+삭제 적용
"결과를 새 시트 '분석결과'로 만들어줘"      → add_sheet 적용
"막대 차트 추가해줘"                        → chart 적용
```

### 편집 결과

- 수정된 파일은 **바탕화면에 자동 저장**됩니다
- 파일명: `원본파일명_edited_날짜시간.xlsx`
- 채팅창에 수행된 작업 요약 + 저장 경로 표시

### Excel 편집 지원 기능 전체 목록

| 기능 | 명령 예시 |
|---|---|
| 셀 값 수정 | "A1 셀을 '총합계'로 바꿔줘" |
| 범위 데이터 입력 | "A2부터 새 데이터 넣어줘" |
| 행 삭제 | "3, 5, 7번 행 삭제해줘" |
| 행 삽입 | "2번 행 위에 새 행 추가해줘" |
| 정렬 | "B열 기준 내림차순 정렬" |
| 수식 입력 | "D2에 =SUM(B2:C2) 넣어줘" |
| 수식 자동 채우기 | "D2 수식을 D10까지 채워줘" |
| 스타일 적용 | "헤더에 스타일 적용해줘" |
| 열 너비 조정 | "A열 너비를 25로 맞춰줘" |
| 틀 고정 | "1행 틀 고정해줘" |
| 시트 이름 변경 | "Sheet1을 '매출데이터'로 바꿔줘" |
| 시트 추가/삭제 | "분석결과 시트 추가해줘" |
| 셀 병합 | "A1:C1 병합해줘" |
| 차트 생성 | "A1:B10 범위로 막대 차트 만들어줘" |

## 9. PC 진단 및 자동 최적화

### 전체 진단 실행

**방법 1**: 채팅창에 명령
```
"PC 상태 확인해줘"
"진단 실행해줘"
"내 컴퓨터 괜찮아?"
```

**방법 2**: 메인 창 → **PC 진단** 탭 → **전체 진단 실행**

### 진단 항목

| 항목 | 확인 내용 |
|---|---|
| 임시 파일 | %TEMP% 폴더 용량 |
| 디스크 여유 공간 | 10% 미만 시 경고 |
| 메모리 사용률 | 85% 초과 시 경고 |
| CPU 온도 | WMI 온도 센서 |
| 시작 프로그램 | 불필요 프로그램 목록 |
| Windows 업데이트 | 미설치 업데이트 확인 |
| 보안 위협 | 의심 프로세스·레지스트리 |

### 자동 최적화 (원클릭)

진단 결과 화면에서 **자동 수리** 버튼을 클릭하거나:

```
"느려졌어, 정리해줘"
"최적화해줘"
"임시 파일 지워줘"
```

수행 작업:
- 임시 파일 삭제 (%TEMP%, Windows Temp)
- 브라우저 캐시 정리
- 불필요 레지스트리 항목 제거
- DNS 캐시 초기화

> **주의**: 레지스트리 청소는 시스템 복원 지점을 만든 후 진행합니다.

- 사람처럼 마우스 이동·스크롤

## 11. 이메일 자동화

### Outlook 연동 (완전 자동화)

Outlook이 설치된 경우 AI가 실제로 메일을 읽고·작성하고·발송합니다.

**받은 메일 확인**
```
"받은 메일 확인해줘"
"오늘 온 메일 요약해줘"
"중요한 메일 있어?"
```

**메일 답장**
```
"홍길동 메일에 답장해줘 — 내일 오후 2시 미팅 가능하다고"
"영업팀 메일에 거절 답장 보내줘"
"미팅 요청 메일 수락해줘"
```

**메일 발송**
```
"김부장님께 보고서 제출 완료 메일 보내줘"
"팀원들한테 내일 오전 10시 회의 공지 메일 발송해줘"
```

**캘린더 연동**
```
"받은 미팅 요청 캘린더에 자동으로 추가해줘"
"이번 주 빈 시간 찾아서 미팅 잡아줘"
```

### 네이버·다음 메일 연동 (IMAP — 읽기·요약)

> ⚠️ IMAP 연동은 **읽기와 요약만** 지원합니다.  
> 자동 발송·답장은 Outlook 연동에서만 가능합니다.

**설정 방법**

1. 메인 창 → **이메일** 탭 → **IMAP 계정 추가**
2. 이메일 주소·비밀번호 입력
3. 네이버: IMAP 서버 `imap.naver.com`, 포트 993
4. 다음: IMAP 서버 `imap.daum.net`, 포트 993

> **네이버 주의**: 네이버 계정 → 보안 설정 → **IMAP/SMTP 사용 허용** 활성화 필요

## 13. Proactive AI (알아서 실행)

Proactive AI는 사용자가 요청하지 않아도 **알아서 감지하고 행동**합니다.

### 자동 브리핑

매일 아침 지정 시간에 자동으로:
- 오늘 날씨 및 기온
- 오늘 캘린더 일정
- PC 상태 요약 (메모리, 디스크)
- 중요 받은 메일 요약

**브리핑 시간 설정**
```
"아침 8시에 브리핑 설정해줘"
"브리핑 꺼줘"
"브리핑 시간 7시 30분으로 바꿔줘"
```

### 자동 이상 감지

24시간 모니터링 중 이상 감지 시 자동 알림:

| 이벤트 | 동작 |
|---|---|
| CPU 온도 85°C 초과 | 알림 + 냉각 권장 |
| 메모리 90% 초과 | 알림 + 메모리 정리 제안 |
| 디스크 여유 5% 미만 | 알림 + 정리 제안 |
| 의심 프로세스 감지 | 즉시 알림 |
| 백신 만료 임박 | 갱신 안내 |

## 14. 워크플로우 자동화

```
"매주 월요일에 지난주 업무 보고서 만들어줘"
"파일 다운로드 폴더가 500MB 넘으면 정리해줘"
```

### 워크플로우 관리

메인 창 → **워크플로우** 탭에서:
- 생성된 워크플로우 목록 확인
- 활성화/비활성화 토글
- 즉시 실행 테스트
- 삭제

### 워크플로우 예시

**예시 1: 일일 자동 보고서**

```
  1. PC 성능 스냅샷 수집
  2. 오늘 처리한 작업 목록 수집
```

**예시 2: 파일 자동 정리**

```
  1. 다운로드 폴더 스캔
```

## 19. 자주 묻는 질문 (FAQ)

**Q. 처음에 캐릭터가 보이지 않아요.**  
A. 시스템 트레이(우측 하단) NEXUS 아이콘을 클릭하거나, `Ctrl+Shift+N` 을 눌러보세요.

**Q. "백엔드 연결 실패" 메시지가 떠요.**  
A. NEXUS를 종료 후 다시 실행하세요. 백그라운드에서 자동으로 서버가 시작됩니다. 방화벽에서 NEXUS를 허용해주세요 (포트 17891).

**Q. API 키를 입력했는데 "API 키 오류"가 나요.**  
A. 키 앞뒤에 공백이 없는지 확인하세요. Perplexity 키는 `pplx-`로 시작합니다.

**Q. 크롤링이 안 되고 "봇 탐지" 오류가 나요.**  
A. 일부 사이트는 탐지가 강화된 경우가 있습니다. 잠시 후 다시 시도하거나, 다른 사이트로 검색해보세요.

**Q. Excel 편집 후 파일이 어디 저장되나요?**  
A. 바탕화면에 `원본파일명_edited_날짜시간.xlsx` 형식으로 저장됩니다.

**Q. Outlook이 없어도 이메일 기능을 쓸 수 있나요?**  
A. 네이버·다음 메일은 IMAP으로 읽기·요약이 가능합니다. 단, 자동 발송·답장은 Outlook 설치가 필요합니다.

**Q. 내 파일이 외부로 유출될 수 있나요?**  
A. NEXUS는 127.0.0.1(로컬)에서만 실행됩니다. PC 파일·이메일·시스템 정보는 외부로 전송되지 않습니다. AI 응답 생성 시 질문 텍스트만 Perplexity/Claude API로 전송됩니다.

**Q. Windows 10에서도 모든 기능이 되나요?**  
A. Windows 10 (버전 1903, 2019년 5월 업데이트 이상)에서 대부분 기능이 지원됩니다. Windows Recall 기능은 Windows 11 전용입니다.

**Q. 프로그램을 완전히 종료하려면?**  
A. 시스템 트레이(우측 하단 시계 근처) NEXUS 아이콘 **우클릭** → **종료**.
