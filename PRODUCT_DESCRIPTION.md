## 한 줄 소개

> **"말 한마디로 PC를 움직이는 AI 팀 비서 — 설치 없이 exe 하나로 시작"**

---

## 제품 개요

Nexus AI는 Windows PC 위에 항상 떠 있는 AI 비서입니다.
음성 명령 또는 텍스트 입력으로 PC 관리·업무 자동화·정보 검색을 한 번에 처리합니다.
단순한 챗봇이 아니라 **PC와 직접 연결된 실행형 AI**입니다.

| 항목 | 내용 |
|------|------|
| 버전 | 2.5.0 |
| 지원 OS | Windows 10 / 11 (64비트) |
| 설치 방법 | 포터블 exe 복사 실행 또는 인스톨러 |
| 필요 사양 | RAM 4GB 이상, 인터넷 연결 (AI 기능) |
| 언어 | 한국어 / 영어 |
| 가격 정책 | 무료 체험 7일 → 월정액 / 영구 라이선스 |

## 핵심 차별점

### 1. 🎭 AI 팀 시스템 (Multi-Persona)
하나의 AI가 아닌 **7명의 전문 AI 팀**이 대기 중입니다.

| 페르소나 | 전문 영역 |
|----------|-----------|
| 🤖 Nexus (기본) | PC 관리·최적화·자동화 전반 |
| 🔬 리서치 Nexus | 경쟁사 분석·시장조사·논문 검색 |
| 💰 재무 Nexus | 예산·ROI·재무 보고서·투자 검토 |
| 🎯 회의 Nexus | 회의 진행·요약·액션 아이템 추적 |
| 🎨 크리에이티브 Nexus | 카피라이팅·아이디어·콘텐츠 기획 |
| 🛡️ 보안 Nexus | 사이버 위협·취약점·침해 대응 |
| ⚖️ 법무 Nexus | 계약서 검토·법률 리스크·Compliance |

> "재무 모드로 바꿔서 이번 달 비용 분석해줘" → 즉시 전환

### 2. ⚡ Auto Workflow Agent
복잡한 여러 단계 작업을 **말 한마디**로 자동 처리합니다.

```
"이번 주 보고서 만들어서 팀장님 메일로 보내줘"
→ ① 파일 검색 → ② 문서 요약 → ③ 보고서 생성 → ④ 이메일 발송
```

### 3. 🧠 Second Brain (장기 기억)
과거 메모·이메일·회의·화면 기록을 모두 인덱싱해서 자연어로 검색합니다.

```
"작년에 내가 만든 경쟁사 분석 자료 찾아줘"
"지난달 팀장님한테 온 이메일에서 예산 관련 내용 찾아줘"
```

### 4. 🎬 Live Caption (실시간 자막·번역)
Zoom·YouTube·웹사이트 어디서든 시스템 오디오를 실시간 전사하고 한국어로 번역합니다.

## 전체 기능 목록

### 🖥️ PC 관리
- CPU·RAM·온도·디스크 실시간 모니터링
- 전체 PC 진단 및 원클릭 수리
- 임시 파일·레지스트리 정리
- 드라이버 점검·Windows Update 확인
- 부팅 속도 분석 및 최적화
- 시스템 복구 포인트 생성

### 🛡️ 보안
- 해킹·원격 접속 탐지
- 의심 프로세스·포트 스캔
- Windows Defender 상태 확인
- VirusTotal 파일 악성코드 검사
- hosts 파일 변조 탐지
- 앱 권한 감사 (카메라·마이크)

### 🤖 AI 자동화
- **Auto Workflow**: 자연어 목표 → 단계별 자동 실행
- **자동화 매크로**: 반복 작업 자연어로 등록·실행
- **자연어 스케줄러**: "매일 오전 9시에 PC 진단해줘"
- **AI 문서 요약·비교**: PDF·Word·Excel 즉시 분석
- **클립보드 AI**: 복사한 내용 자동 요약·번역·교정

### 🎙️ 음성·회의
- 웨이크워드 감지 ("헤이 넥서스")
- **회의 녹음·전사·요약**: 회의 중 녹음 → AI 요약 + 액션 아이템 자동 추출
- **음성 받아쓰기**: 말하면 현재 활성 앱에 자동 타이핑
- **실시간 자막·번역**: 해외 영상·회의 한국어 실시간 자막

### 🧠 기억·검색
- **Second Brain**: 메모·이메일·화면·회의 통합 검색
- **Windows Recall**: 60초 주기 화면 캡처·OCR·키워드 검색
- **장기 메모리**: 대화 내용 자동 학습·기억
- **파일 심층 검색**: 파일 내용까지 검색 (PDF·Word 내부 텍스트)
- **중복 파일 탐지·정리**

### 📅 업무 연동
- **Outlook 캘린더**: 일정 확인·추가 음성 명령
- **Outlook 이메일**: 받은 편지함 확인·AI 요약·발송
- **업무 일지 자동 생성**: 오늘 한 일 자동 정리
- **Excel 데이터 내보내기**
- **Google/Naver 가격 비교**·뉴스 수집

### 🌐 외부 연동
- **날씨·교통**: 실시간 날씨 + 목적지 이동 시간
- **스마트홈**: Home Assistant 연동 (조명·에어컨 음성 제어)

- **브라우저 자동화**: 웹사이트 자동 탐색·데이터 수집

### 📊 리포트·알림
- **PC 건강 리포트**: 일간·주간 PDF 생성 및 이메일 발송

## 설치 방법

### 방법 A — 포터블 (설치 없이 즉시 실행) ⭐ 추천

```
1. Nexus_2.5.0_portable.exe 다운로드
2. 원하는 폴더에 복사 (바탕화면, USB 등)
3. 더블클릭 → 실행
4. 완료! 설치·관리자 권한 불필요
```

### 방법 B — 인스톨러 (기업·장기 사용자)

```
1. Nexus_2.5.0_setup.exe 다운로드
2. 더블클릭 → 다음 → 설치
3. 바탕화면 바로가기 생성 → 실행
```

> ⚠️ Windows 10/11 64비트 전용입니다.

## 초기 설정 (1분 완료)

1. **첫 실행** → 온보딩 화면
2. **AI 이름·성별 설정** (예: "넥서스", 여성 목소리)
3. **Groq API 키 입력** (무료, groq.com에서 발급)
4. **완료** → 즉시 사용 가능

> 💡 Groq API는 무료 티어로 일반 사용에 충분합니다.
> 고급 기능(GPT-4급): Claude API 키 선택 입력

## 자주 묻는 질문

**Q. 인터넷 없이도 사용할 수 있나요?**
PC 관리 기능(진단·정리·보안·모니터링)은 오프라인에서 100% 작동합니다.
AI 대화·번역·요약 등 LLM 기능은 인터넷 연결이 필요합니다.

**Q. 개인정보가 외부로 나가나요?**
PC 내부 데이터(파일·화면·이메일)는 사용자 PC에만 저장됩니다.
AI 질의 시 입력한 텍스트만 Groq/Claude API로 전송되며, 학습에 사용되지 않습니다.

**Q. 백신·보안 프로그램이 차단하나요?**
일부 백신이 PC 제어 기능(PowerShell 실행)을 경고할 수 있습니다.
신뢰 프로그램으로 예외 등록하시면 됩니다.

**Q. Outlook이 없어면 캘린더·이메일 기능이 작동하나요?**
Outlook 미설치 시 해당 기능은 비활성화됩니다.
나머지 모든 기능은 정상 작동합니다.
